# 使用框架

* 服务注册: nacos
* 服务熔断: sentinel
* 服务网关: spring cloud gateway
* 服务调用: openfeign
* 鉴权: spring security
* 消息队列: rabbitmq(使用spring cloud stream)
* 数据库: mysql
* 数据库连接池: druid
* 缓存: redis
* 分布式事务: seata
